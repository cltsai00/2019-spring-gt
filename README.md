# Graph-Theory-Project
Graph Theory (EE6622E) in National Cheng Kung University.
-------
## 目的
-------
#### 在無向圖中，找出中國郵差問題的解

-------
## 解題邏輯
-------
#### 1. 計算各點的degree，找出圖中odd degree的點，同時將各條edge雙向連接(視為無向)
#### 2. 對odd degree的node做matching
#### 3. 找出matching的node的最短路徑
#### 4. 將找出的路徑在接上一條無向edge
#### 5. 找到Euler cycle即為解
